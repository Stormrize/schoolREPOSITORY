# schoolREPOSITORY

## Git

### Herkunft

Git ist ein dezentrales Versionskontrollsystem, welches von Linus Torvals im Rahmen des Linux Projektes entwickelt wurde.

### Befehle

Um ein leeres Repositry zu erstellen schreibt man git init

Um datein hinzufügen schreibt man git add <Datei oder . für alles>

Um Änderungen zu speichern schreibt man git commit

Um lokale Änderungen auf remote Repository zu übertragen schreibt man git push

Um Änderungen aus dem remote Repository zu nehmen schreibt man git pull

Um ein Repo (lokal oder remote) zu kopieren verwende git copy <Verzeichnisname (lokal) ODER https, ssh oder github (remote)>

Um die Staging Area (sozusagen Zwischenspeicher wo Änderungen vor commits gespeichert werden müssen) (geht auch für eine einzelne Datei) zurückzusetzen verwende git restore <Datei oder . für alles>

### Fazit

Git ist ein wertvolles Tool für die Versionierung und findet mit der Softwareentwicklung (und ihren meist textbasierten Formaten) eine populäre Anwendung.
